# hello-vinod Vite React App

Follow the steps below to run this project on your local machine and see the `hello vinod` heading in the browser.

## Prerequisites
- [Node.js](https://nodejs.org/) 18 or newer (comes with npm)

## Getting started
1. **Clone or download the project**
   ```bash
   git clone <repository-url>
   cd gigaml_projects
   ```
   > If you downloaded a ZIP, unzip it and open a terminal in the project directory instead.

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```
   The terminal will show a local URL (usually `http://localhost:5173`). Open it in your browser to view the app.

4. **Stop the server**
   Press `Ctrl + C` in the terminal when you are finished.

## Project structure
- `index.html` – HTML entry point that loads the React app
- `src/main.jsx` – React entry file that renders the `App` component
- `src/App.jsx` – Component displaying the "hello vinod" heading
- `vite.config.js` – Vite configuration
- `package.json` – Scripts and dependencies

## Helpful scripts
- `npm run dev` – Run the app in development mode
- `npm run build` – Create an optimized production build in `dist/`
- `npm run preview` – Preview the production build locally

Enjoy experimenting with React!
